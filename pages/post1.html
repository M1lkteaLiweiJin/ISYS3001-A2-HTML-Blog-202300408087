Post 1: Getting Started with HTML Blog Development

Published on: September 20, 2025 | Author: Student ID | Category: Technical Notes

This article is a test post for the ISYS3001 Assignment 2 assessment, documenting the development process of a static HTML blog with the following key steps:

1. Page Structure Design

The blog system includes a homepage (article list), article detail pages, and an about page. It uses modular design with consistent styling through CSS to ensure visual consistency. The homepage displays article titles and excerpts using unordered lists, with clickable titles that link to detail pages.

2. Style Optimization

An external CSS file (style.css) is used to manage styles, with 1.5 line spacing (meeting assessment requirements) and responsive layout for different devices like computers and mobile phones. The navigation bar uses a dark background, and article cards have shadow effects to enhance user experience.

3. Version Control and Deployment

A repository was created using GitHub's web interface, implementing a "main+feature/blog-dev" branch strategy. After development, branches were merged and deployed to GitHub Pages for free online access.
